Asset Bank: Asset Panels Prototype

  • This prototype will not yet display correctly in Internet Explorer. Please use Chrome, Firefox or Safari.
  • These panels will appear when browsing and searching Asset Bank.
  • You can select the assets, add/remove from lightbox and click through to a dummy detail page.
  • The download links do not do anything, but show how extra functionality could be incorporated.
  • This prototype does not include the larger image that appears on hover - but you can assume that will continue to behave as it already does.

Please have a play with both options below and indicate which you prefer at the bottom of the page.
